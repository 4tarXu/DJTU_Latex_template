# DJTU_Latex_template

(非官方) 大连交通大学本科生毕业设计(论文)相关latex模版，模版内校徽、校名来源于官网"视觉系统"。

## 📚 模版简介

本仓库提供了大连交通大学本科生毕业设计(论文)相关的 LaTeX 模版集合，适用于不同专业和不同文档类型的需求。所有模版均按照学校官方格式要求制作，方便学生快速生成规范化的毕业设计文档。

## 📁 模版结构

本仓库包含以下4个模版：

### 1. (R信管)DJTU_Latex_综合毕业设计(论文)

**适用对象：** 信息管理专业综合毕业设计论文
**文档内容：** 封面、中文摘要、英文摘要、正文、谢辞、参考文献

### 2. (R信管)DJTU_Latex_外文翻译

**适用对象：** 信息管理专业外文翻译
**文档内容：** 封面、正文

### 3. (R信管)DJTU_Latex_调研报告

**适用对象：** 信息管理专业调研报告
**文档内容：** 封面、正文

### 4. (物流)DJTU_Latex_毕业设计(论文)

**适用对象：** 物流专业毕业设计论文
**文档内容：** 封面、中文摘要、英文摘要、正文、谢辞、参考文献

## 📂 模版目录结构

每个模版均采用统一的目录结构：

```
模版名称/
├── main.tex                    # 主文件（学生无需修改）
├── latex/                      # 内容文件目录（学生需要编辑）
│   ├── cover.tex              # 封面信息
│   ├── Chapter.tex            # 正文内容
│   ├── cn_abstract.tex        # 中文摘要（部分模版含）
│   ├── en_abstract.tex        # 英文摘要（部分模版含）
│   └── Thanks.tex             # 谢辞（部分模版含）
├── bib/                        # 参考文献目录
│   └── ref.bib                # 参考文献文件
└── figure/                     # 图片资源目录
    ├── logo.png               # 校徽
    ├── logo.pdf               # 校徽PDF版本
    ├── djtu.jpg               # 校名
    └── example.png            # 示例图片
```

## 🚀 使用步骤

### 第一步：选择模版

根据你的专业和文档类型，选择合适的模版文件夹。

### 第二步：编辑封面信息

打开 `latex/cover.tex` 文件，修改以下个人信息：

```latex
\newcommand\thesisTitle{XXX系统的分析与实现}       % 题目
\newcommand\thesisAuthor{姓名}                       % 姓名
\newcommand\thesisAuthorNum{学号}                    % 学号
\newcommand\thesisClass{专业班级}                    % 专业班级
\newcommand\supervisor{指导教师}                     % 指导教师
\newcommand\supervisortitle{职称}                    % 指导教师职称
\newcommand\thesisDate{完成日期}                     % 完成时间
```

### 第三步：编写正文内容

在 `latex/Chapter.tex` 文件中编写你的论文正文。模版提供了以下环境：

- **章节：** 使用 `\chapter{}` 创建章
- **小节：** 使用 `\section{}` 创建节
- **子节：** 使用 `\subsection{}` 创建小节

### 第四步：添加中英文摘要

如果模版包含摘要部分，在 `latex/cn_abstract.tex` 和 `latex/en_abstract.tex` 中分别编写中英文摘要。

### 第五步：添加谢辞

如果模版包含谢辞部分，在 `latex/Thanks.tex` 中编写谢辞内容。

### 第六步：管理参考文献

在 `bib/ref.bib` 文件中添加参考文献，引用格式使用 BibTeX 标准。在正文中引用参考文献时使用 `\cite{}` 命令。如果是**非实引文献，则使用 `\nocite{}`命令。

### 第八步：编译文档

#### 使用 TexPage（推荐）

1. 将所有文件压缩成 `.zip`格式上传到 Texpage 项目
2. 点击“编译"按钮完成编译，并下载PDF文件。

#### 使用本地 TeX 编辑器

使用 XeLaTeX 编译器编译（建议使用 TeX Live 2019 或更高版本）：

**Visual Studio Code + LaTeX Workshop：**

- 安装 LaTeX Workshop 插件
- 使用 XeLaTeX 编译

**Texstudio：**

- 菜单：`选项` → `配置 Texstudio` → `构建`
- 将默认编译器设置为 `XeLaTeX`

**命令行编译：**

```bash
xelatex main.tex
biber main
xelatex main.tex
xelatex main.tex
```

> 💡 **提示：** 所有模版均需要使用 **XeLaTeX(2019+)** 进行编译，建议关闭拼写检查功能以避免不必要的干扰。

## ⚙️ 模版特性

- ✅ **符合学校格式：** 严格按照大连交通大学本科生毕业设计格式要求制作
- ✅ **中英文支持：** 完美支持中英文混排，使用 `ctex` 宏包
- ✅ **自动目录：** 自动生成目录，支持三级标题
- ✅ **参考文献：** 使用 BibLaTeX + Biber，支持 GB/T 7714-2015 国家标准
- ✅ **图表管理：** 自动编号和交叉引用，编号格式为"章号-序号"
- ✅ **双面打印：** 自动处理奇偶页边距和页眉页脚

## 📝 注意事项

1. **不要修改主文件：** `main.tex` 文件包含所有格式设置，学生无需修改
2. **保持目录结构：** 不要随意移动或删除文件，保持目录结构完整
3. **图片格式：** 建议使用 PNG、JPG 或 PDF 格式的图片
4. **编码设置：** 所有 `.tex` 文件使用 UTF-8 编码
5. **编译顺序：** XeLaTeX → Biber → XeLaTeX → XeLaTeX（通常编辑器会自动处理）
