## 1. 页面设置

### 1.1 基础设置

**纸张：** A4  
**方向：** 纵向  
**装订线：** 无  

### 1.2 页边距

**操作路径：** `页面布局` → `页边距` → `自定义边距`

| 位置 | 数值 | 说明 |
|------|------|------|
| 上 | 25mm（2.5cm） | 上边距 |
| 下 | 25mm（2.5cm） | 下边距 |
| 左 | 25mm（2.5cm） | 左边距 |
| 右 | 25mm（2.5cm） | 右边距 |
| 装订线 | 0mm | 不使用装订线 |

**注意：**
- 以上为单页设置
- 如需双面打印，需在"页码范围"的"多页"选项中设置为"对称页边距"

### 1.3 页眉页脚距离

**操作路径：** `页面布局` → `页边距` → `自定义边距` → `版式`

| 位置 | 数值 | 说明 |
|------|------|------|
| 页眉顶端距离 | 15mm（1.5cm） | 页眉与页面上边缘的距离 |
| 页脚底端距离 | 17.5mm（1.75cm） | 页脚与页面下边缘的距离 |

### 2.1 正文字体

**设置位置：** `开始` → `字体`

| 元素 | 字体 | 字号 | 字体样式 |
|------|------|------|---------|
| 中文 | 宋体 | 12pt | 常规 |
| 英文/数字 | Times New Roman | 12pt | 常规 |

**注意：**
- 12pt = 小四号（实际为 10.5pt，但Word中显示为12pt）

## 3. 段落格式

### 3.1 正文段落

**操作路径：** `开始` → `段落` → 右下角箭头 → 打开"段落"对话框

| 设置项 | 数值 | 说明 |
|--------|------|------|
| **缩进** |
| 左侧 | 0 字符 | 左缩进 |
| 右侧 | 0 字符 | 右缩进 |
| 特殊格式 | 首行缩进 | 首行缩进 |
| 磅值 | 2 字符 | 缩进两个字符 |
| **间距** |
| 段前 | 0 磅 | 段落前间距 |
| 段后 | 0 磅 | 段落后间距 |
| **行距** |
| 行距 | 固定值 | 行距类型 |
| 设置值 | 20 磅 | 固定行距 |

**重要提示：**
- ✅ 行距必须设置为"固定值 20 磅"
- ✅ 不要使用"单倍行距"、"1.5倍行距"等
- ✅ 首行缩进为 2 个字符
- ✅ 段落前后间距均为 0

│ 提高了课程排课的准确度和效率，为学   │
│ 校工作人员节省了大量宝贵的时间。通   │

## 5. 正文格式

### 5.1 正文要求

- **字体：** 宋体
- **字号：** 12pt（小四号）
- **行距：** 固定值 20 磅
- **首行缩进：** 2 个字符
- **段前/段后：** 0 磅
- **对齐方式：** 两端对齐

### 5.2 特殊字符处理

- 中文字符之间：自动间距
- 中文与英文/数字之间：自动加半角空格
- 章节标题中的空格：可用全角空格或通过插入特殊符号实现

### 5.3 正文示例

```
现如今，各式教务系统在各大高校得到了越来越广泛地应用，这些系统大大
提高了课程排课的准确度和效率。通过系统操作，我们可以轻松地得到合理
的排课安排，并且系统能够将大量的课程信息进行有效地整合。

教学辅助软件在现代教育中发挥着越来越重要的作用。尤其是课表转换和分
析功能，为学校、教师和学生提供了许多便利和优势。
```

**注意：**
- 段落之间用空行分隔
- 不要使用回车键强制换行
- 让 Word 自动处理换行和排版

## 6. 图表格式

### 6.1 图片格式

#### 6.1.1 图片标题

| 设置项 | 数值 |
|--------|------|
| 位置 | 图片下方 |
| 字体 | 宋体 |
| 字号 | 五号（10.5pt） |
| 对齐方式 | 居中 |
| 编号格式 | 图 X-Y（章号-图序号） |
| 标题与图间距 | 10pt（0.35cm） |

**示例：**
```
      [图片内容]
      
    图 2-1  系统业务流程图
（宋体五号，居中，距离图10pt）
```

#### 6.1.2 图片插入步骤

1. 在正文中定位插入位置
2. `插入` → `图片` → 选择图片文件
3. 调整图片大小，使其适应页面宽度
4. 图片`对齐方式`设置为`居中`
5. 在图片下方插入标题
6. 标题格式：宋体五号，居中

#### 6.1.3 图片与正文间距

- **图片前间距：** 适当空行（建议1-2个空行）
- **图片后间距：** 适当空行（建议1-2个空行）

**示例布局：**
```
正文内容，系统业务流程图如图2-1所示。

[空行]

        [图片]
     
     图 2-1  系统业务流程图

[空行]

正文内容，通过以上流程可以清楚地了解...

```

### 6.2 表格格式

#### 6.2.1 表格标题

| 设置项 | 数值 |
|--------|------|
| 位置 | 表格上方 |
| 字体 | 宋体 |
| 字号 | 五号（10.5pt） |
| 对齐方式 | 居中 |
| 编号格式 | 表 X-Y（章号-表序号） |
| 标题与表间距 | 0pt（紧密贴合） |

**示例：**
```
    表 2-1  示例表格
（宋体五号，居中，距离表0pt）

┌──────┬────────────────┐
│ 本质 │ 过程           │
├──────┼────────────────┤
│ 途径 │ 规划、实施     │
└──────┴────────────────┘
```

#### 6.2.2 三线表制作

**步骤：**
1. `插入` → `表格` → 选择行列数
2. 选中整个表格，右键 → `边框和底纹`
3. 设置为"三线表"：
   - 上线：粗线（2.25磅）
   - 中线：细线（0.5磅）
   - 下线：粗线（2.25磅）
   - 其他线：无

**三线表示意图：**
```
┌─────────────┬─────────────┐
│   列标题1   │   列标题2   │  ← 顶线（2.25磅）
├─────────────┼─────────────┤
│   数据1     │   数据2     │  ← 中线（0.5磅）
│   数据3     │   数据4     │
├─────────────┼─────────────┤
│   合计      │   合计值    │  ← 底线（2.25磅）
└─────────────┴─────────────┘
```

#### 6.2.3 表格内文字

| 设置项 | 数值 |
|--------|------|
| 字体 | 宋体 |
| 字号 | 五号（10.5pt） |
| 行距 | 1.5倍（相对于五号字） |
| 对齐方式 | 水平居中，垂直居中 |

**操作：**
1. 选中表格内所有文字
2. `开始` → `字体`：宋体五号
3. `开始` → `段落` → `行距`：1.5倍行距
4. `表格工具` → `布局` → `对齐方式`：居中

#### 6.2.4 表格与正文间距

- **表格前间距：** 适当空行
- **表格后间距：** 适当空行

**示例布局：**
```
正文内容，相关数据如表2-1所示。

[空行]

    表 2-1  示例表格

┌──────┬────────┐
│ ...  │  ...   │
└──────┴────────┘

[空行]

正文内容，根据表格数据分析得出...

```

### 6.3 图表引用

在正文中引用图表时：

**格式：** "如图X-Y所示" 或 "如表X-Y所示"

**示例：**
- "师生课表自动生成系统整体业务流程构造如图2-1所示："
- "相关数据如表2-1所示，"

**注意：**
- 图、表的引用号与文字之间通常有一个空格
- 引用位置应在图表出现之前或附近

## 7. 页眉页脚

### 7.1 页眉设置

#### 7.1.1 正文页眉

| 设置项 | 数值 |
|--------|------|
| 字体 | 黑体 |
| 字号 | 小五号（9pt） |
| 对齐方式 | 居中 |
| 内容 | "大连交通大学 XX 届本科生综合毕业设计（论文）" |

**操作步骤：**
1. `插入` → `页眉` → `编辑页眉`
2. 在页眉区域输入文本
3. 选中文本，设置：黑体，小五号，居中
4. 双击正文区域退出页眉编辑

**注意：**
- XX 届需要根据实际情况替换（如 2026 届）
- 页眉只在正文部分显示（从第一章开始）
- 封面、摘要、目录部分不显示页眉

#### 7.1.2 摘要页眉

**摘要页面：**
- ❌ 不显示页眉内容（页眉空白）
- ✅ 只显示页码

### 7.2 页脚设置

| 设置项 | 数值 |
|--------|------|
| 字体 | 宋体 |
| 字号 | 小五号（9pt） |
| 对齐方式 | 居中 |
| 内容 | 页码 |

**操作步骤：**
1. `插入` → `页码` → `页面底端` → `普通数字2`（居中）
2. `开始` → `字体`：宋体，小五号

### 7.3 页码格式

#### 7.3.1 页码类型

| 文档部分 | 页码格式 | 起始页码 |
|---------|---------|---------|
| 封面 | 无页码 | - |
| 摘要 | 大写罗马数字（I, II, III...） | 从 I 开始 |
| 目录 | 续摘要页码 | 连续 |
| 正文 | 阿拉伯数字（1, 2, 3...） | 从 1 开始 |
| 参考文献、谢辞 | 续正文页码 | 连续 |

#### 7.3.2 页码设置步骤

**摘要部分（罗马数字）：**
1. 定位到摘要第一页的页脚
2. `插入` → `页码` → `设置页码格式`
3. 编号格式：选择"Ⅰ, Ⅱ, Ⅲ..."
4. 起始页码：Ⅰ

**正文部分（阿拉伯数字）：**
1. 定位到第一章第一页的页脚
2. `插入` → `页码` → `设置页码格式`
3. 编号格式：选择"1, 2, 3..."
4. 起始页码：1

### 9.2 目录内容

### 9.3 目录自动生成

**步骤：**
1. 确保所有章节标题使用了Word样式（标题1、标题2、标题3）
2. 定位到要插入目录的位置（摘要之后）
3. `引用` → `目录` → `自动目录`

## 13. 常见问题与解决方法

### 13.1 行距问题

**问题：** 设置固定值 20 磅后行距过大或过小

**解决：**
1. 检查字号设置是否为 12pt
2. 确保行距类型选择了"固定值"而不是"单倍行距"或"多倍行距"
3. 检查是否有段落间距影响

### 13.2 首行缩进问题

**问题：** 首行缩进不正确

**解决：**
1. 选中段落 → `开始` → `段落` → 特殊格式选择"首行缩进"
2. 磅值输入"2 字符"或"28.35 磅"（约等于2个字符）
3. 确保未使用空格手动缩进

### 13.3 页眉页脚问题

**问题：** 页眉页脚格式不一致

**解决：**
1. 双击页眉/页脚进入编辑模式
2. 选中所有页眉/页脚内容
3. 统一设置字体、字号
4. 检查不同节之间的页眉页脚链接

### 13.4 图表编号问题

**问题：** 图表编号自动更新错误

**解决：**
1. 使用Word的题注功能
2. `引用` → `插入题注`
3. 设置标签（"图"或"表"）
4. 编号格式选择"包含章节号"

### 13.5 页码格式问题

**问题：** 摘要和正文页码格式混乱

**解决：**
1. 在摘要和正文之间插入分节符
2. `页面布局` → `分隔符` → `下一页`
3. 分别设置各节的页码格式
4. 正文部分取消"链接到前一节"

## 14. Word操作技巧

### 14.1 快速设置全文格式

1. 全选文档（Ctrl+A）
2. 设置基础格式：宋体，12pt，固定值 20 磅
3. 逐段检查并调整标题格式

### 14.2 使用样式设置标题

1. `开始` → `样式` → 修改"标题1"、"标题2"、"标题3"
2. 设置格式后应用到相应标题
3. 便于后续生成目录

### 14.3 批量替换

1. `开始` → `替换`（Ctrl+H）
2. 可批量替换标点符号、空格等

### 14.4 防止孤行孤字

1. `开始` → `段落` → `换行和分页`
2. 勾选"孤行控制"
3. 勾选"段前分页"（对于章标题）

### 14.5 表格自动调整

1. 选中表格
2. `表格工具` → `布局` → `自动调整`
3. 选择"根据窗口自动调整表格"
